## 1. 登录与账户

### 1.1 登录

1. 打开浏览器，访问 `http://localhost:5173`（或生产域名）
2. 输入**邮箱**和**密码**
3. 勾选「记住我」可将登录有效期延长至 30 天（默认 2 小时）
4. 点击「登 录」

> ⚠️ 连续输入错误密码会导致账户临时锁定，请联系管理员重置。

### 1.2 登出

点击右上角头像区域，或直接在个人设置中选择登出。

## 3. 内容管理

> 路径：侧栏 → 内容体系 → **内容管理**

### 3.1 内容列表

- 支持按**标题关键词**搜索
- 支持按**状态**筛选（草稿 / 已发布 / 归档）
- 支持按**类型**筛选（文章 / 页面 / 公告）
- 每条内容显示：标题、类型标签、状态徽标、作者、浏览量、发布时间

**操作按钮**：
| 按钮 | 说明 |
|------|------|
| 编辑 | 进入内容编辑页 |
| 发布 | 将草稿直接发布（仅草稿状态可见）|
| 取消发布 | 将已发布内容退回草稿（仅发布状态可见）|
| 删除 | 软删除（可恢复）|

---

### 3.2 创建内容

点击右上角**「+ 新建内容」**进入创建页面。

**左栏 — 主体内容**：

| 字段 | 说明 |
|------|------|
| 标题 * | 内容标题，输入后自动生成 Slug |
| Slug * | URL 友好标识符（只允许小写字母、数字、连字符）|
| 摘要 | 可选，显示在列表页的简短描述（最多 500 字）|
| 正文 * | Markdown 编辑器，支持实时预览 |

**Markdown 编辑器使用技巧**：
- 工具栏提供：粗体、斜体、删除线、分割线、引用、代码块、图片、表格、列表等快捷按钮
- 编辑器支持左右分栏预览模式
- 可点击全屏按钮进入专注写作模式

**右栏 — 元数据**：

| 字段 | 说明 |
|------|------|
| 内容类型 * | 文章 / 页面 / 公告 |
| 分类 | 选择内容分类（可选）|
| 封面图片 | 上传或选择图片 URL |
| 定时发布 | 选择未来时间自动发布（留空则点击发布时立即生效）|
| SEO 标题 | 自定义搜索引擎标题（可选）|
| SEO 描述 | 自定义搜索引擎描述（最多 160 字）|

**底部操作**：
- **保存草稿** — 保存为草稿，不公开显示
- **立即发布** — 保存并立即发布
- **取消** — 放弃更改，返回列表

---

### 3.3 编辑内容

在列表页点击「编辑」，进入与创建相同的表单（已预填内容）。

底部按钮变为：
- **保存草稿** — 保存但不改变发布状态
- **保存并发布** — 保存并设为已发布

## 4. 分类管理

> 路径：侧栏 → 内容体系 → **分类管理**

支持树形层级分类。

### 创建分类

1. 点击「新建分类」
2. 填写：
   - **分类名称** *（必填）
   - **Slug** *（必填，URL 标识）
   - **描述**（可选）
   - **父分类**（可选，不选则为顶级分类）
   - **排序值**（数值越小越靠前）
3. 点击「创 建」

### 编辑 / 删除

列表中点击对应按钮。注意：**有子分类的分类无法删除**，需先删除所有子分类。

## 5. 标签管理

> 路径：侧栏 → 内容体系 → **标签管理**

标签用于内容的扁平化分类，不支持层级。

### 创建标签

1. 点击「新建标签」
2. 输入**标签名称**，Slug 会**自动生成**（可手动修改）
3. 点击「创 建」

列表显示**使用量**，可用于了解哪些标签最受欢迎。

## 6. 评论管理

> 路径：侧栏 → 内容体系 → **评论管理**

### 状态 Tab

| Tab | 说明 |
|-----|------|
| 全部 | 所有评论 |
| 待审核 | 用户提交等待审核的评论 |
| 已通过 | 已批准公开显示的评论 |
| Spam | 被标记为垃圾评论 |

### 操作

- **批量选择** — 勾选左侧复选框
- **批量审核通过** — 选中后点击「批量通过」
- **批量删除** — 选中后点击「批量删除」
- 单条操作：通过 / 标记 Spam / 删除

## 7. 媒体库

> 路径：侧栏 → 内容体系 → **媒体库**

用于管理所有上传的图片、视频和文档文件。

### 上传文件

1. 点击「上传文件」按钮，或**直接拖放文件**到上传区域
2. 支持格式：JPG、PNG、GIF、WebP、PDF、MP4
3. 单文件最大 **10MB**

### 筛选

通过顶部下拉框按类型筛选（全部 / 图片 / 视频 / PDF）。

### 预览与删除

- 图片文件悬停后显示预览大图
- 点击删除图标删除文件（不可恢复）

## 8. 用户管理（管理员）

> 路径：侧栏 → 用户体系 → **用户管理**  
> ⚠️ 需要 `admin` 角色权限

### 用户列表

显示所有用户，包含：用户名、邮箱、昵称、角色标签、状态、最后登录时间。

### 编辑用户

点击「编辑」可修改：
- 昵称
- 邮箱
- 账号状态（正常 / 禁用）
- 分配角色（多选）

### 删除用户

软删除，用户数据保留但无法登录。**不能删除自己的账户**。

## 9. 角色管理（管理员）

> 路径：侧栏 → 用户体系 → **角色管理**

## 13. 操作日志

> 路径：侧栏 → 系统 → **操作日志**  
> ⚠️ 只读，不可修改或删除

记录所有用户的关键操作，用于审计和问题排查。

### 筛选

通过顶部下拉框按**操作类型**筛选：
- `USER_LOGIN` / `USER_LOGOUT` — 登录/登出

## 常见问题

**Q: 内容保存后不见了？**  
A: 检查内容是否保存为草稿，列表默认显示所有状态，可用"全部状态"筛选框确认。

**Q: 上传图片失败？**  
A: 检查：① 文件格式是否为 JPG/PNG/GIF/WebP；② 文件大小是否超过 10MB；③ 网络连接是否正常。

**Q: 没有权限访问用户管理？**  
A: 需要管理员为您的账户分配 `admin` 角色，请联系系统管理员。

**Q: 如何恢复已删除的内容？**  
A: 当前版本删除操作为软删除，可联系开发人员通过数据库操作恢复。后续版本将提供回收站功能。

## 16. 影视模块（管理员）

入口：左侧菜单「影视管理」。

### 16.1 影片列表

- 支持按**类型**（电影 / 电视剧 / 动漫 / 综艺 / 短剧）、**地区**、**年份**、**状态**筛选
- 支持搜索片名、原名
- 列表展示封面、评分、连载状态、最近更新时间

### 16.2 新增 / 编辑影片

必填字段：

| 字段 | 说明 |
|------|------|
| 标题 | 中文片名 |
| Slug | URL 别名，留空将自动从标题生成 |
| 类型 | movie / tv / anime / variety / short |
| 状态 | 草稿 / 已发布 |

可选字段：原名、海报、评分、地区、年份、是否完结、总集数、当前集数、简介、导演、主演、标签等。

### 16.3 播放线路与集数

进入影片详情后切换「线路与集数」标签：

1. 新增线路（如「线路 1」「线路 2」），可填备注
2. 在线路下批量添加集数：每集填集数序号、标题、播放地址（m3u8 / mp4）、可选时长
3. 也支持批量粘贴格式 `1集$URL\n2集$URL`，由后端解析

### 16.4 注意事项

- 播放地址需确保前端可跨域访问，推荐 HLS（`.m3u8`）
- 删除线路会同时删除其下所有集数

## 17. 小说模块（管理员）

入口：左侧菜单「小说管理」。

### 17.1 作品列表

支持按分类、连载状态（连载 / 完结 / 暂停）、状态筛选，搜索书名/作者。

### 17.2 新增 / 编辑作品

必填：标题、Slug、分类。  
可选：作者、封面、简介、连载状态、评分、标签。

字数和章节数会在保存章节时自动累计，无需手动维护。

### 17.3 章节管理

进入作品详情 → 「章节」标签：

1. 「新增章节」：填写章节序号、标题、正文（富文本编辑器）
2. 章节列表支持拖拽排序
3. 删除章节会自动重算父作品的字数和章节数

## 18. 漫画模块（管理员）

入口：左侧菜单「漫画管理」。

### 18.1 作品列表

支持按分类、连载状态筛选，搜索漫画名 / 作者。

### 18.2 新增 / 编辑作品

必填：标题、Slug、分类。  
可选：作者、封面、简介、连载状态、评分、标签。

### 18.3 话节管理

每一话由「话序号 + 标题 + 多张图片」组成：

1. 新增话节，填写话序号、标题
2. 在「图片」区域上传或粘贴图片 URL（保存为 JSON 数组 `pageUrls`）
3. 拖拽调整图片顺序
4. 保存后前台漫画阅读页将按顺序纵向展示

## 19. 前台站点（Portal）使用

前台地址：`http://localhost:3002`（生产替换为正式域名）。

### 19.1 访问入口

| 路径 | 说明 |
|------|------|
| `/` | 首页：聚合最新文章 / 影视 / 小说 / 漫画 |
| `/articles` | 文章列表（支持分类、标签筛选） |
| `/articles/[slug]` | 文章详情 + 评论 |
| `/movies` | 影视列表（类型 / 地区 / 年份 筛选 + 分页） |
| `/movies/[slug]` | 影视详情，可选择线路、集数，HLS 播放 |
| `/novels` | 小说列表 |
| `/novels/[slug]` | 小说详情 + 目录 |
| `/novels/[slug]/chapters/[id]` | 章节阅读页（上一章 / 目录 / 下一章） |
| `/comics` | 漫画列表 |
| `/comics/[slug]` | 漫画详情 + 话目录 |
| `/comics/[slug]/chapters/[id]` | 漫画阅读页（多图纵向滚动） |
| `/categories/[slug]` | 分类聚合 |
| `/tags/[slug]` | 标签聚合 |

### 19.2 评论

- 文章详情底部「发表评论」无需登录，填写昵称、邮箱、内容即可提交
- 若后台「评论审核」开启，新评论需在管理后台审核通过后才公开显示
- 网站名称/Logo/ICP/是否启用评论 等通过后台「系统配置 → 站点设置」控制，前台首屏拉取 `/site-settings/public`

### 19.3 缓存策略

前台基于 Next.js ISR：

| 类型 | revalidate |
|------|-----------|
| 列表页 | 30 秒 |
| 详情页 | 60 秒 |
| 章节内容 | 300 秒 |
| 评论 | 不缓存（每次实时拉取） |

如需立即看到改动，可在后台触发对应内容的更新或重启 portal 进程。

## 20. Demo 数据 Seed 脚本

用于本地或新环境快速填充演示数据。

```bash
cd backend
node scripts/seed-demo.js
```

将插入：

- 3 个分类（技术 / 生活 / 资讯）
- 3 篇示例文章
- 3 部小说，每部 5 章
- 3 部漫画，每部 3 话，每话 4 张占位图（picsum）
- **12 部影视**（v1.0.3 新增）：覆盖 5 种 movieType + 4 个地区
- **每篇文章 3-5 条评论**（v1.0.3 新增）：含嵌套回复 + 1 条 pending
- **9 个站点配置键**（v1.0.3 新增）：site_name / enable_comment 等
- 为已存在的所有影片补充「线路 1」+ Apple BipBop 测试 m3u8

脚本幂等：按 slug / 复合键检查，重复执行不会产生脏数据。

**影视线路（lib/movie-utils.ts）**：新增 `friendlySourceName(name, index)` — 包含 CJK 字符视为可读名称直接展示；否则视为内部采集代号，统一显示为「线路 N」。详情页和播放页均已替换。`movieTypeLabel()` 同步提取为公共函数。

**阅读体验（app/globals.css）**：
- 字体栈新增 `Microsoft YaHei UI`（Windows 渲染更清晰，排在 YaHei 之前）
- `.novel-reader`：衬线/黑体混合字体栈、1.95 行距、首行缩进 2em、字号 17px
- `.comic-page-wrap`：包裹漫画图片，防止懒加载导致的 CLS；`decoding="async"` 加速解码

**导航（components/Header.tsx）**：移除「后台登录」按钮（原指向 localhost:5173），避免在生产环境暴露管理后台入口。
